# Real-Time-Smile-Detection
(i) Developed a real-time smile detection system using OpenCV and Python, achieving ~90% face detection accuracy and processing 25–30 frames per second via webcam input.
(ii) Implemented Haar Cascade-based facial and smile recognition with optimized parameters, reducing false positives by ~35% and improving detection consistency across multiple faces (up to 3–5 simultaneously).
(iii) Optimized image processing pipeline using ROI (Region of Interest) techniques, decreasing computational load by ~40% and enhancing real-time performance efficiency.
